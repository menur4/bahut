# Politique de Confidentialité - Bahut

**Dernière mise à jour : 2 février 2026**

## Introduction

Bahut est une application mobile permettant aux élèves et parents de consulter les notes scolaires via l'API École Directe. Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos données.

## Données collectées

### Données d'authentification
- Identifiants École Directe (nom d'utilisateur et mot de passe)
- Token de session temporaire

### Données scolaires
- Notes et moyennes
- Emploi du temps
- Devoirs
- Informations de vie scolaire (absences, retards)

### Données techniques
- Préférences de l'application (thème, notifications)
- Statistiques d'utilisation anonymes

## Utilisation des données

Vos données sont utilisées exclusivement pour :
- Afficher vos informations scolaires
- Calculer vos moyennes et statistiques
- Synchroniser avec votre calendrier (si activé)
- Envoyer des notifications locales (si activées)

## Stockage des données

- **Stockage local uniquement** : Toutes vos données sont stockées sur votre appareil
- **Identifiants sécurisés** : Vos identifiants sont chiffrés via Flutter Secure Storage
- **Aucun serveur tiers** : Nous ne possédons pas de serveur. L'application communique directement avec l'API École Directe

## Partage des données

Nous ne partageons, vendons ou transférons aucune donnée personnelle à des tiers. Les seules communications réseau sont :
- Avec l'API École Directe (pour récupérer vos données scolaires)

## Sécurité

- Chiffrement des identifiants stockés
- Option d'authentification biométrique (Face ID / empreinte digitale)
- Aucune donnée transmise à des serveurs tiers

## Droits des utilisateurs

Vous pouvez à tout moment :
- Supprimer vos données en vous déconnectant de l'application
- Désinstaller l'application pour supprimer toutes les données locales

## Mineurs

Cette application est destinée aux élèves et parents. Les mineurs doivent avoir l'autorisation de leurs parents ou tuteurs légaux pour utiliser l'application.

## Modifications

Cette politique peut être modifiée. Les utilisateurs seront informés des changements significatifs via une mise à jour de l'application.
